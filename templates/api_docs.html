API de Datos del Agente Autónomo

Esta API permite a otras IAs y sistemas automatizados acceder y utilizar los datos recolectados por el agente autónomo de Ibiza.

Nota: Todos los endpoints devuelven datos en formato JSON a menos que se especifique lo contrario.

Endpoints Disponibles

GET /api/data

Obtiene datos del agente autónomo con filtros opcionales.

Parámetros de Consulta
Nombre Tipo Descripción Requerido Ejemplo
category string Filtrar por categoría. Usar "all" para todas las categorías. No (por defecto: "all") events
limit integer Número máximo de resultados por categoría. No 10
source string Filtrar por fuente de datos. No ibiza-tourism
date_from string (ISO date) Filtrar desde una fecha específica. No 2023-01-01
date_to string (ISO date) Filtrar hasta una fecha específica. No 2023-12-31
Ejemplos de Solicitud

Obtener todos los datos:

GET /api/data

Obtener datos de eventos con límite:

GET /api/data?category=events&limit=5
Respuesta
{
  "success": true,
  "api_info": {
    "version": "1.0",
    "documentation": "/api/docs",
    "filters_applied": {
      "category": "events",
      "limit": 5,
      "source": null,
      "date_from": null,
      "date_to": null
    }
  },
  "data": {
    "events": [
      {
        "timestamp": "2023-06-15T14:30:00",
        "source": "ibiza-tourism",
        "data": {
          "name": "Fiesta en Pacha",
          "location": "Pacha Ibiza",
          "date": "2023-07-01",
          "description": "Gran fiesta de verano en la discoteca Pacha"
        }
      },
      // Más eventos...
    ]
  }
}

GET /api/download/{format}/{category}

Descarga datos de una categoría específica en el formato solicitado.

Parámetros de Ruta
Nombre Tipo Descripción Requerido Ejemplo
format string Formato de descarga. Opciones: json, csv, txt Sí json
category string Categoría de datos a descargar. Sí events
Ejemplos de Solicitud

Descargar datos de eventos en formato JSON:

GET /api/download/json/events

GET /api/download/{format}/all

Descarga todos los datos en el formato solicitado.

Parámetros de Ruta
Nombre Tipo Descripción Requerido Ejemplo
format string Formato de descarga. Opciones: json, txt Sí json
Ejemplos de Solicitud

Descargar todos los datos en formato JSON:

GET /api/download/json/all

POST /api/force_collection

Fuerza una nueva recolección de datos del agente autónomo.

Cuerpo de la Solicitud

No requiere cuerpo.

Ejemplos de Solicitud
POST /api/force_collection
Respuesta
{
  "success": true,
  "message": "Se ha iniciado la recolección de datos en segundo plano"
}

Integración con IAs

Esta API está diseñada para ser fácilmente consumida por otras IAs. Los datos están estructurados de manera que las IAs puedan procesarlos eficientemente.

Ejemplo de uso con Python y requests
import requests
import json

# Obtener datos de la API
response = requests.get('http://localhost:5000/api/data?category=events&limit=10')
data = response.json()

if data['success']:
    # Procesar los datos con la IA
    events = data['data']['events']
    
    for event in events:
        print(f"Evento: {event['data'].get('name')}")
        print(f"Fecha: {event['data'].get('date')}")
        print(f"Fuente: {event['source']}")
        print("---")
else:
    print(f"Error: {data.get('error')}")
Importante: Esta API no requiere autenticación en este momento ya que está diseñada para uso local. Si se expone a Internet, se recomienda implementar autenticación.
